# 🌙 Selamat Idul Fitri 1447 H

Website ucapan Selamat Hari Raya Idul Fitri 1447 Hijriyah yang dibuat dengan desain modern, elegan, dan interaktif. Dibuat oleh **Kemas Alfath**


## ✨ Fitur Utama

### 🎨 Visual & Animasi
- **Preloader** animasi bulan sabit saat loading
- **Confetti** emas & hijau yang bertebaran saat halaman dimuat
- **Floating elements** — bintang, bulan, dan ornamen melayang di background
- **Parallax scrolling** — efek kedalaman saat scroll
- **Scroll progress bar** — indikator posisi scroll di atas halaman
- **Glassmorphism** — efek kaca modern pada card dan elemen

### 📝 Konten
- **Hero Section** — Foto profil dengan ucapan personal & kaligrafi Arab
- **Greeting Cards** — 6 kartu ucapan dengan animasi scroll
- **Quote Section** — Kutipan Al-Quran (QS. Al-Insyirah: 6-7)
- **Closing Section** — Pesan penutup dengan foto profil

### 🎮 Interaktif
- **"Maafin Saya Ya?"** — Fitur interaktif dengan mekanik tombol yang lucu:
  - Tombol "Tidak" akan mengecil setiap kali diklik
  - Tombol "Iya" akan membesar secara progresif
  - Setelah 5x menolak, tombol "Tidak" hilang total!
  - Gambar kucing lucu minta maaf 🐱
  - Efek confetti saat akhirnya dimaafkan 🎉

### 🔊 Audio
- Background musik takbiran (toggle on/off)
- Tombol audio di pojok kanan bawah

## 🚀 Cara Menjalankan

### Lokal
```bash
# Clone repository
git clone https://github.com/kemasmalfath/project_ied.git

# Masuk ke folder project
cd project_ied

# Jalankan dengan http-server (atau live server lainnya)
npx http-server . -p 8080

# Buka di browser
# http://localhost:8080
```

### Deploy
Website ini bisa di-deploy ke platform hosting statis manapun:
- **Netlify** — Drag & drop folder di [app.netlify.com/drop](https://app.netlify.com/drop)
- **Vercel** — Import dari GitHub
- **GitHub Pages** — Aktifkan di Settings → Pages
- **Cloudflare Pages** — Connect repository GitHub
